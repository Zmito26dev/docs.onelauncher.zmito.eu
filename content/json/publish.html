Publish JSON Content ​

Publishing via npoint ​

The simplest and free way to publish your JSON Content is through npoint. The main feature of npoint is that it's a JSON endpoint which allows you to modify the JSON file at any time. This is ideal for us since we need to update our Content instance at any moment.

To do this, you can follow these steps:

  1. Create an account on npoint.
  2. Create a new JSON bin and paste all your JSON Content.
  3. Save it and at the bottom you'll be able to get the link to that JSON binary.

This link is all you need to activate your Content instance

TIP

Instead of entering the complete link when activating a new Content you can replace https://api.npoint.io/<id> with npoint-<id> and convert it into a key. It will work the same way!

Publishing via another provider ​

You can use any JSON file host you want. You only need the public link of the JSON binary for it to work.

All set! ​

Once you have the link or the key of your Content instance you can activate it from ONE Launcher. Once activated and downloaded you can start playing without any problem.
